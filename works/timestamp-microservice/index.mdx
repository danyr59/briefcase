# [Timestamp Microservice ](https://timestamp-microservice-d.herokuapp.com/ "timestamp-microservice") 

timestamp microservice, this small server-side application, made for educational purposes, in the
absolute understanding and construction of REST API.

The application consists of making get queries through this endpoint:

```
GET /api/2015-12-25
```

*o*

```
GET /api/1451001600000"
```

and return a json object like this:

```js
{ 
  "unix":1451001600000,
  "utc":"Fri, 25 Dec 2015 00:00:00 GMT"
}
```
